# Asteroids

'roids in the terminal, for fun and practice.

[![asciicast](https://asciinema.org/a/pQW1YaWq0O27kvp0R4SexP4gc.png)](https://asciinema.org/a/pQW1YaWq0O27kvp0R4SexP4gc)

## Playing

```
$ ruby asteroids.rb
```

Controls:

- **w** : Thrust
- **s** : Fire
- **a** : Turn Left
- **d** : Turn Right
- **q** : Quit
